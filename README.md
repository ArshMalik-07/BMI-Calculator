# 💪 BMI Calculator

A simple and responsive BMI Calculator built using HTML, CSS, and JavaScript.

---

## 🚀 Features
- Calculate BMI instantly
- Shows category (Underweight, Normal, Overweight, Obese)
- Clean UI

---

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript

---

## ▶️ How to Run
1. Clone the repo
2. Open index.html in browser
